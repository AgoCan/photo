# 记录笔记

纽摄：追求好的画面的基本原则
1. 一幅好照片要有一个鲜明的主题（有时也称之为主体）。或者表现一个人，或是表现一件事物，甚至可以表现该题材的一个故事情节。主题必须明确，毫不含糊，使任何观赏者一眼能看得出来。
2. 一幅好照片必须能把注意力引向被摄主体，换句话说，使观赏者的目光一下子投向被摄主体。
3. 一幅好照片必须画面简洁，指包括有利于把视线引向被摄主体的内容，而排除或压缩哪些可能分散注意力的内容
